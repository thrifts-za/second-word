# Application logo assets

The public simulator uses locally served SVG logo marks so application context
is recognisable without a runtime CDN dependency. The files were downloaded
from the corresponding Worldvectorlogo entries on 2026-07-22:

- [Official Gmail Icon (2020)](https://worldvectorlogo.com/logo/official-gmail-icon-2020-)
- [Slack new logo](https://worldvectorlogo.com/logo/slack-new-logo)
- [Microsoft Teams](https://worldvectorlogo.com/logo/microsoft-teams-1)
- [WhatsApp](https://worldvectorlogo.com/logo/whatsapp-3)
- [X](https://worldvectorlogo.com/logo/x-2)
- [LinkedIn icon](https://worldvectorlogo.com/logo/linkedin-icon)

The marks identify simulated third-party writing surfaces. Their owners do not
sponsor or endorse Second Word. Second Word's quotation-peel mark is separate
and remains the only custom product identity.
