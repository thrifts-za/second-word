# Second Word submission evidence

The deployed 60-case evaluation records both whether a verified passage appeared and whether the exact draft earned a signed rewrite credential. All six gracious drafts that had already met their moment received zero rewrite offers; nine of fifteen weighty drafts retained an optional rewrite when their own words needed care.

The model can add care for contextual signals such as sarcasm, while visible local signals remain an independent OR condition. Scripture still comes only from YouVersion, Guide never receives a rewrite credential, and two contract-invalid model responses fail closed to application-authored silence.
